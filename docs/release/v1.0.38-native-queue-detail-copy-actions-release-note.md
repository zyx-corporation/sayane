# Sayane v1.0.38 — Native Queue Detail Copy Actions

## Summary

Sayane `v1.0.38` adds direct copy actions to the native macOS queue detail screen so operators can
copy the current candidate detail, diff, and lineage without manual text selection.

## Highlights

- queue detail now shows dedicated copy buttons for candidate detail, diff, and lineage
- clipboard export uses the same localized detail labels already shown in the native UI
- docs now describe queue detail as a practical handoff/export surface in addition to review

## Boundary

- this release does not add profile mutation beyond the existing review actions
- this release does not export a separate file artifact; it copies the current review text to the
  clipboard only
