# Sayane v1.0.39 — Native LaunchAgent Copy Actions

## Summary

Sayane `v1.0.39` adds direct copy actions to the native macOS LaunchAgent detail groups so the
current state and recovery preview can be moved into handoff notes without manual selection.

## Highlights

- `Current State Details` now copies the visible LaunchAgent state summary
- `Recovery Preview Details` now copies the current runtime-init, cleanup, and repair preview text
- native docs now describe the LaunchAgent detail groups as copy-friendly troubleshooting surfaces

## Boundary

- this release does not add any new mutation path
- this release copies text only; it does not create a file export
