# v1.0.45 Native Daemon Handoff Debug Shell and Next Action Release Note

## Summary

The native macOS daemon handoff export now includes the debug-shell URL and the first next-action summary near the top of the saved note.

## What changed

- added `Debug Shell` to the exported handoff note
- added the first `Next Command` and its `Reason` near the phase summary header
- kept the export local-first, plain-text, and read-first

## Operator impact

- browser-side troubleshooting now starts from the exported note without reopening diagnostics
- the first recommended daemon action is visible immediately instead of only inside lower sections
- handoff recipients can orient faster before scanning the full note

## Validation

- `swift build --package-path macos/SayaneApp`
- focused `swift test --package-path macos/SayaneApp --filter ...` still intermittently stalls after test bundle link through `swiftpm-xctest-helper` in this environment
