# v1.0.72 P3 Explicit Review-Queue Import Release Note

## Summary

Sayane `v1.0.72` closes the remaining P3 package-import gap by adding an
explicit review-queue import command for vault-aware packages.

## Included work

- `sayane storage queue-package` persists pending candidates from one verified
  vault-aware package
- `sayane storage import-package` remains preview-only and non-mutating
- queue import stays separate from profile mutation and candidate approval

## Boundary retained

- queue import does not merge into the profile
- candidate approval remains explicit and separate
- legacy Obsidian / Git compatibility paths remain outside the primary path
