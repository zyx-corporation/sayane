# Sayane v1.1 Release Closure: Scoped Context Acceptance and Safe Reuse

**Status**: closed_with_notes
**Date**: 2026-06-07

---

## Summary

Sayane v1.1 extends the v1.0 Context Acceptance architecture by adding safeguards for partial context reuse.

The central risk addressed in this release is **Trojan context**: locally valid context that loses its scope or conditions and is later treated as global profile context.

v1.1 introduces `scoped_accept`, preserves scope and conditions through MCP output, surfaces scoped context in review views, previews scoped context in external packages, and makes scope leak / promotion risk visible in the transfer regression dashboard.

---

Sayane v1.1 は、v1.0 の文脈受容アーキテクチャを、部分文脈の安全な再利用へ拡張する。

このリリースが扱う中心リスクは、**トロイの木馬的文脈混入**である。局所的には妥当な文脈が、scope や condition を失い、後続処理で全体プロファイルのように扱われる危険を防ぐ。

v1.1 では `scoped_accept` を導入し、MCP出力・候補確認・package preview・HTML dashboard まで、scope / conditions / negative_constraints / promotion risk を保持・表示する。

## Boundary Statements

- `scoped_accept` is not ordinary approval.
- Scoped context must not be promoted to global context without explicit review.
- MCP output is derived context, not canonical profile.
- Package Preview is not package import.
- Verified package is not automatic acceptance.
- Dashboard is observation, not enforcement.
- T-RDE is not an automatic truth oracle.
- Sayane remains a human-review-centered context acceptance system.

### 境界宣言（日本語）

- 範囲付き許容 (`scoped_accept`) は、通常の採用 (`approve`) ではない。
- scope付き文脈は、明示的な再レビューなしに全体文脈へ昇格してはならない。
- MCP出力は正本プロファイルではなく、対象ツール向けの派生文脈である。
- Package Preview は import ではない。
- 検証済みpackageは自動受容を意味しない。
- Dashboard は観測面であり、制御面ではない。
- T-RDE は自動的な真理判定装置ではない。
- Sayane は、人間の確認を中心に置く文脈受容基盤である。

## CLI Summary (v1.1)

```bash
# Scoped context acceptance
sayane review scoped-accept <id> --scope project:target:subscope \
  --conditions "condition1,condition2" \
  --negative "constraint1,constraint2" --reason "..."

# MCP compiled context with scope preservation
sayane context-compile --target cursor --mode full|compact|strict
sayane context-compile --target cursor --show-scope

# Package import preview (no profile modification)
sayane package preview pkg/

# Transfer regression HTML dashboard
sayane transfer-report --format html -o dashboard.html
```
