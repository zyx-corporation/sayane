# v1.0.41 Native Daemon Handoff Export Release Note

## Summary

The native macOS daemon screen now exports one combined plain-text handoff note from the app header.

## What changed

- added `handoff noteを書き出す` to the native daemon header
- exports a single `.txt` file containing:
  - operator summary rail
  - phase-closure gates
  - suggested actions
  - read surfaces
  - LaunchAgent current state
  - LaunchAgent recovery preview
- keeps the export local-first and read-first by writing only the current daemon screen-state summary

## Operator impact

- daemon-side handoff no longer depends on multiple copy actions when a single note is enough
- the operator can save the current daemon context directly to disk before ending a session
- the exported file keeps troubleshooting and next-step context together in one artifact
