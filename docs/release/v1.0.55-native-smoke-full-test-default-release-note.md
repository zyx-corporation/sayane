# v1.0.55 Native Smoke Full Test Default Release Note

## Summary

- switched `scripts/check-macos-app-preview.sh` to run the full
  `swift test --package-path macos/SayaneApp --disable-xctest` suite by default
- removed the old filtered smoke path from the script because it remained
  unreliable in this environment
- aligned README / getting-started guidance with the current SwiftPM behavior

## Why

- the `xctest` bridge still hangs intermittently here, while direct Swift Testing
  execution completes cleanly and gives a truer smoke signal for the native app
  surface
- focused filtered runs still depend on SwiftPM filter semantics that are not a
  reliable default in this checkout

## Operator impact

- `./scripts/check-macos-app-preview.sh` now validates the same package-wide
  regression surface we use for normal native app verification
- the smoke path now stays on one stable command instead of branching between
  `xctest` and Swift Testing behaviors

## Validation

- `swift build --package-path macos/SayaneApp`
- `swift test --package-path macos/SayaneApp --disable-xctest`
