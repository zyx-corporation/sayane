# Sayane v1.0.37 — Native Direct Preview Launch

## Summary

Sayane `v1.0.37` upgrades the macOS preview launcher so the native app can be launched directly from
the built Swift executable without requiring Xcode as the default path.

## Highlights

- `scripts/run-macos-app-preview.sh` now starts or reuses the local Bridge, then launches the built
  native preview executable directly
- the launcher can still fall back to Xcode with `--xcode`
- the launcher can reuse the current build with `--no-build` or stay attached to the terminal with
  `--foreground`
- docs now describe direct native preview launch as the primary local verification path

## Boundary

- this release does not package a signed `.app` bundle yet
- this release does not replace the Bridge-backed local contract
- this release does not change the app's operator/read-only boundary

## Recommended reads

```bash
./scripts/run-macos-app-preview.sh
./scripts/run-macos-app-preview.sh --no-build
./scripts/run-macos-app-preview.sh --foreground
./scripts/run-macos-app-preview.sh --xcode
```
