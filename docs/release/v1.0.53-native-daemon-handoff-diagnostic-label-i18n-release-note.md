# v1.0.53 Native Daemon Handoff Diagnostic Label I18N Release Note

## Summary

The native macOS daemon handoff export now localizes more diagnostic labels for Japanese operators.

## What changed

- localized handoff-facing Japanese labels including status diagnostics, proof diagnostics, return code, profile, token/log file labels, and bridge/source metadata labels
- kept the export structure unchanged while reducing mixed-language headings in Japanese mode
- aligned the handoff note with the same `AppStrings` localization path as the rest of the native app

## Operator impact

- Japanese handoff notes read more consistently without abrupt English headings
- support and handoff text now matches the surrounding native app vocabulary more closely
- future wording tweaks stay centralized in one string table

## Validation

- `swift build --package-path macos/SayaneApp`
- focused `swift test --package-path macos/SayaneApp --filter ...` still executes unreliably in this environment
