# v1.0.47 Native Daemon Handoff Preflight and Proof Entry Release Note

## Summary

The native macOS daemon handoff export now includes preflight and proof-diagnostics entry commands.

## What changed

- added a `Preflight Checks` block to the exported handoff note
- added a `Proof Diagnostics` block to the exported handoff note
- kept the export local-first, plain-text, and read-first

## Operator impact

- exported notes now include the main read-first entrypoints for proof and preflight verification
- handoff recipients can start from app, browser, or CLI diagnostics without reconstructing the command set
- daemon diagnosis now has a more complete first-pass checklist in one saved artifact

## Validation

- `swift build --package-path macos/SayaneApp`
- focused `swift test --package-path macos/SayaneApp --filter ...` still executes unreliably in this environment
