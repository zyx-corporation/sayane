# v1.0.43 Native Daemon Handoff Timestamped Artifact Release Note

## Summary

The native macOS daemon handoff export now produces a timestamped artifact name and captures Bridge version/source metadata in the saved note.

## What changed

- changed the exported filename from a fixed name to `sayane-daemon-handoff-YYYYMMDD-HHMMSS.txt`
- added `Bridge Version` and `Source Updated` lines to the exported handoff note when health data is available
- kept the export format plain text and local-first

## Operator impact

- saved handoff files are easier to sort and compare across repeated sessions
- release and troubleshooting handoff notes now show both runtime context and source freshness
- operators no longer need to rename exported notes by hand to avoid collisions

## Validation

- `swift build --package-path macos/SayaneApp`
- focused `swift test --package-path macos/SayaneApp --filter ...` still intermittently stalls after test bundle link via `swiftpm-xctest-helper` in this environment
