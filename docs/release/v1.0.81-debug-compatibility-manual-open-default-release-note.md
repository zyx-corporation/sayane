# v1.0.81 Debug Compatibility Manual-Open Default Release Note

Sayane continues the macOS-side `/app/ui*` retirement by making the retained
debug compatibility shell manual-open by default.

## Included

- `scripts/run-app-local.sh` no longer opens the browser automatically by default
- explicit browser opening now requires `--open`
- current docs now describe the compatibility shell as a manual fallback instead
  of a nearby Chrome-first path

## Current boundary

- `/app/ui*` remains available for explicit debug, smoke, and handoff work
- this slice does not remove the compatibility shell itself
- native app startup and recovery remain the routine macOS operator path

## Verification used in this slice

```bash
bash -n scripts/run-app-local.sh scripts/run-macos-app-preview.sh scripts/check-macos-app-preview.sh
swift test --package-path macos/SayaneApp
```
