# v1.0.42 Native Daemon Handoff Evidence Metadata Release Note

## Summary

The native macOS daemon handoff export now includes enough Bridge metadata to serve as a small evidence snapshot.

## What changed

- added generated-at timestamp to the exported daemon handoff note
- added current `Profile`, `Bridge URL`, and `Health Endpoint`
- added the current Bridge status headline and detail line
- kept the export format plain text and local-first

## Operator impact

- saved handoff notes now show when they were generated and under which local Bridge context
- the exported note is more useful as a release-side or troubleshooting breadcrumb
- app-to-doc handoff keeps working without opening the browser shell or copying diagnostics separately

## Validation

- `swift build --package-path macos/SayaneApp`
- focused `swift test --package-path macos/SayaneApp --filter ...` remains intermittently blocked by
  `swiftpm-xctest-helper` in this environment after the test bundle links
