# v1.0.71 P3 Vault-Aware External Package Preview Release Note

## Summary

Sayane `v1.0.71` adds the first supported vault-aware external package flow for
P3.

This slice introduces a reviewable package path that is distinct from the
legacy Obsidian / Git compatibility workflows.

## Included work

- `sayane storage export-package` creates a vault-aware external package
- exported `bundle.yml` now carries provenance metadata and content hash
- package manifests now declare `package_kind` and non-canonical review
  boundary metadata
- package manifests now declare artifact-level retention classes and package
  review-window guidance
- package manifests now declare supported operator actions and forbidden
  workflows
- package manifests now declare `import-package` as a preview-only contract
- `sayane storage import-package` verifies a package and previews reviewable
  candidates without mutating the profile
- optional package audit export is redacted JSON

## Boundary retained

- package import is preview-only in this slice
- verified package contents are still not auto-accepted
- Obsidian import / export and Git commit remain legacy compatibility paths

## Remaining P3 work

- none within the current Community P3 boundary; follow-on work moves to later phases
