# v1.0.76 curl+bash Operator Quickstart

Status: handoff_ready

## Purpose

This is the shortest supported operator path for the current Community line.

Start with **`curl | bash` for CLI / Bridge runtime**, then add the native
macOS app only if the operator actually needs the native shell.

## 1. Install

```bash
curl -fsSL https://raw.githubusercontent.com/zyx-corporation/sayane/main/scripts/install.sh | bash
sayane --version
```

Expected outcome:

- `sayane` is available on PATH, or the installer prints the PATH addition hint
- `~/.sayane/` exists after first init
- the current supported runtime is local-only

## 2. Start the local Bridge

```bash
sayane serve
```

Default host/runtime boundary:

```text
http://127.0.0.1:38741
```

Useful first checks:

```bash
TOKEN=$(cat ~/.sayane/bridge.token)
curl -s http://127.0.0.1:38741/health
curl -s -H "Authorization: Bearer $TOKEN" http://127.0.0.1:38741/app/overview
curl -s -H "Authorization: Bearer $TOKEN" http://127.0.0.1:38741/app/contract
```

## 3. Optional native macOS app

Only if the operator wants the native macOS shell:

```bash
./scripts/run-macos-app-preview.sh
```

Or build/install the local bundle:

```bash
bash scripts/build-macos-app-bundle.sh --zip
bash scripts/install-macos-app.sh --no-build
bash scripts/refresh-macos-app.sh --no-build
bash scripts/verify-macos-app-install.sh
open ~/Applications/SayaneApp.app
```

Current boundary:

- native app is the primary operator UI on macOS once installed
- current primary install path still remains `curl | bash`
- native app still expects the local CLI / Bridge runtime

## 4. Debug-only browser compatibility shell

Use only for debugging, smoke, compatibility checks, or handoff:

```bash
open -a "Google Chrome" "http://127.0.0.1:38741/app/ui?bootstrap_token=$(cat ~/.sayane/bridge.token)"
```

Do not use:

```text
http://127.0.0.1:8008/index.html
```

## 5. When something goes wrong

Check in this order:

1. `sayane --version`
2. `curl -s http://127.0.0.1:38741/health`
3. `cat ~/.sayane/bridge.token`
4. `sayane app daemon-operator-phase-status --json`
5. `bash scripts/verify-macos-app-install.sh` if the native app is involved
6. only if native diagnostics are not enough, open the debug-only compatibility shell

## 6. Escalation boundary

Current Community line does **not** yet imply:

- signed pkg delivery
- notarized dmg delivery
- OS service installation closure
- background supervision closure

Those remain later maintainer or future-release steps.
