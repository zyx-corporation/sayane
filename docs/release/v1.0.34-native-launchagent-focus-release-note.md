# Sayane v1.0.34 — Native LaunchAgent Focus

## Summary

Sayane `v1.0.34` extends the native macOS daemon screen by adding a compact `LaunchAgent Focus`
block at the top of the `LaunchAgent` section.

This still does **not** close the packaging and supervision phase. It stays read-first, but now
lets operators scan current state, recovery previews, and the next suggested command before
dropping into the denser detailed groups below.

## Highlights

- native macOS `LaunchAgent` section now starts with a compact focus block
- the focus block groups current state, recovery preview summaries, and the next suggested command
- the denser current-state, preview, command-deck, and Runbook groups remain available below without losing detail
- getting-started, handoff, and native README guidance now describe the LaunchAgent section as having a first-glance entry point

## Boundary

- this release does not add mutating service lifecycle UI
- this release does not change the current `cli_first_local_bridge` supported operator path
- this release does not change any underlying command or status payload; it only adds a compact first-glance layer in the native UI
