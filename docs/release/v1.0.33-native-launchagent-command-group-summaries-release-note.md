# Sayane v1.0.33 — Native LaunchAgent Command Group Summaries

## Summary

Sayane `v1.0.33` extends the native macOS daemon screen by adding short intent summaries to each
`LaunchAgent` command group.

This still does **not** close the packaging and supervision phase. It stays read-first, but now
lets operators understand the purpose of each group before scanning the command list itself.

## Highlights

- native macOS `LaunchAgent` command groups now show one-line summaries before their commands
- inspect, recover, start, and log groups now explain intent in the same inspect-first operator flow
- the summaries reduce the need to infer group purpose from command names alone
- getting-started, handoff, and native README guidance now describe the deck as both grouped and explained

## Boundary

- this release does not add mutating service lifecycle UI
- this release does not change the current `cli_first_local_bridge` supported operator path
- this release does not change any command ordering or command contents; it only adds short explanatory summaries
