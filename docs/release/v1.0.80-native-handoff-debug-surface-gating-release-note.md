# v1.0.80 Native Handoff Debug Surface Gating Release Note

Sayane continues the macOS-side `/app/ui*` retirement by removing the retained
debug compatibility URL from routine native handoff/export output.

## Included

- native daemon handoff export now omits the debug compatibility URL during
  routine healthy flow
- the same URL still appears when explicit debug/session failure conditions make
  the compatibility surface relevant
- maintainers can still force inclusion with
  `SAYANE_INCLUDE_DEBUG_COMPATIBILITY_IN_HANDOFF=1`

## Current boundary

- `/app/ui*` remains available as a maintainer/debug compatibility surface
- native diagnostics and handoff artifacts now prefer backend URL, health,
  token/log paths, and next-command guidance first
- this does not remove the compatibility surface itself
