## Community Edition v1.0.3

### Added

- **Local Open WebUI compile adapter** — `sayane compile --target local-openwebui`
- **PyPI package [`sayane`](https://pypi.org/project/sayane/)** — `pip install sayane==1.0.3` ([#82](https://github.com/zyx-corporation/sayane/issues/82))

### Changed

- Extension **0.3.6–0.3.8**: Open WebUI active insert + URL matching fixes

### Install

```bash
pip install sayane==1.0.3
sayane init
sayane serve
```

Chrome Extension: build from `extension/` (not on PyPI).

**Full changelog**: [CHANGELOG.md](https://github.com/zyx-corporation/sayane/blob/v1.0.3/CHANGELOG.md)
