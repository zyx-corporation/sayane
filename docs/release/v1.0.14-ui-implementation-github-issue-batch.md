# v1.0.14 UI Implementation GitHub Issue Batch

Status: planning_ready

## Goal

Provide one copy-paste batch document for creating the resident app implementation issues in GitHub.

## Shared labels

Base labels for all issues:

- `area:resident-app`
- `type:ui`
- `boundary:app-facing`

Phase labels:

- issues 1-8: `phase:mvp`
- issues 9-14: `phase:production`

Additional labels:

- issues 7 and 14: `area:daemon`
- issues 9 and 12: `type:infra`
