# v1.0.54 Native Daemon Handoff CLI Label I18N Release Note

## Summary

The native macOS daemon handoff export now localizes more CLI-adjacent labels through `AppStrings`.

## What changed

- localized `Component` to `コンポーネント` in Japanese mode
- localized the `Tail Commands` section heading to `ログ追跡コマンド`
- moved `launchctl print`, `stdout tail`, and `stderr tail` handoff labels under the app string table

## Operator impact

- Japanese handoff notes now read more consistently through the CLI diagnostics portion
- the export relies less on hard-coded mixed-language labels
- future terminology adjustments for these labels now stay centralized in the same localization surface

## Validation

- `swift build --package-path macos/SayaneApp`
- focused `swift test --package-path macos/SayaneApp --filter ...` still executes unreliably in this environment
