# v1.0.52 Native Daemon Handoff Bridge Context I18N Release Note

## Summary

The native macOS daemon handoff export now localizes the Bridge Context section header through the app string table.

## What changed

- moved the Bridge Context heading into `AppStrings`
- localized the Japanese heading to `Bridgeコンテキスト`
- kept the export structure unchanged while aligning the header with the app's i18n policy

## Operator impact

- the handoff note uses the same localization path as the rest of the native app strings
- the opening section no longer relies on a hard-coded English label in Japanese mode
- future wording adjustments can now be made from one localization surface

## Validation

- `swift build --package-path macos/SayaneApp`
- focused `swift test --package-path macos/SayaneApp --filter ...` remains unreliable in this environment
