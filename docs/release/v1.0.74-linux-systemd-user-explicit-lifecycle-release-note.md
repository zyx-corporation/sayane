# Sayane v1.0.74 — Linux systemd --user Explicit Lifecycle Slice

## Summary

Sayane extends the Linux `systemd --user` shipped slice from preview/apply/status
into explicit local lifecycle control.

## What changed

- added `sayane app daemon-systemd-user-daemon-reload --json`
- added `sayane app daemon-systemd-user-enable-now --json`
- added `sayane app daemon-systemd-user-disable-now --json`
- upgraded Linux operator metadata so the shipped slice now includes reviewed
  unit write plus explicit local reload/enable/disable control

## Boundaries

This release still does not close:

- generic install/remove/update lifecycle automation
- Linux desktop packaging
- background supervision shipment
